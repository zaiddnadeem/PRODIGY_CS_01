# PRODIGY_CS_01

## Caesar Cipher Encryption Tool

This project is part of my Cyber Security Internship at Prodigy InfoTech.

### Description
A Python-based Caesar Cipher tool that allows users to encrypt and decrypt messages using a custom shift value.

### Features
- Encrypt messages
- Decrypt messages
- Custom shift values
- Supports uppercase and lowercase letters
